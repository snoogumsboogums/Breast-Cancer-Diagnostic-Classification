This project uses the Breast Cancer Wisconsin (Diagnostic) Dataset, which contains diagnostic data of breast cancer obtained from digitized images of fine needle aspirate (FNA) of breast mass biopsies.

The dataset consists of numerical features describing cell nuclei characteristics, such as radius, texture, perimeter, and area. It is used to build a classification model that predicts whether a tumor is malignant (cancerous) or benign (non-cancerous) based on these extracted features.
